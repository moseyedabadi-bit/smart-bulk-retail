# Smart Bulk Dispenser Showcase

This section presents the evolution of the Smart Bulk Dispenser platform.

The product roadmap includes three implementation levels:

1. Level 1 – Manual Gravity Dispenser
2. Level 2 – Smart Dispenser
3. Level 3 – POS Integrated Smart Dispenser

Each level is designed to address different operational, financial and technological requirements.

The framework allows retailers to start with a low-cost solution and progressively upgrade to a fully integrated retail platform.
